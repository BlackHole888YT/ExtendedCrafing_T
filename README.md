# Extended Crafting Table

A custom Minecraft 1.21.1 mod that adds highly configurable crafting recipes with RPG-style conditions.

- 🇬🇧 **English Documentation** → [guide_en.md](README_EN.md)

## Features

- Custom shaped recipes
- Player health requirements
- Experience cost (points & levels)
- Time of day conditions
- Dimension restrictions
- Advancement requirements
- Tool durability consumption
- Custom enchanted book support
- Minecraft 1.21.1 Data Components support
